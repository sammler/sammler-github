## sammler-strategy-github

> Sammler strategy to collect GitHub content.

## Install

First install yarn:

```
npm install -g yarn
```

**Production environment**

```sh
$ yarn run docker-compose
```

**Development environment**

```sh
$ yarn run dc-dev-up
```
